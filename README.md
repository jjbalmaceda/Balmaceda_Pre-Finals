What is Generative Ai?
This website is about Generative AI (GenAI), a technology that can create text, images, music, and other content using artificial intelligence. It helps users understand how GenAI works, its applications, ethical considerations, and provides articles and FAQs to guide responsible use.
 Instructions to Run the Project
 Download or clone the repository** to your local computer:  
   ```bash
   git clone <your-repository-link>
Open the project folder on your computer.

Open the HTML files in a web browser:

index.html – Home page

faq.html – Frequently Asked Questions

article.html – Articles about GenAI

ethics.html – Ethical use of GenAI
You can double-click the files or right-click → Open with → Browser.

Optional: Use a code editor like VS Code with the Live Server extension to preview the website live.
